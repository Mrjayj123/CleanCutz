# Clean Cutz | AI Video Studio

Clean Cutz is a high-performance, browser-based video editing interface designed with a modern "Glassmorphic" aesthetic. It allows you to browse cinematic footage via the Pexels API and perform precise "virtual cuts" using Media Fragment technology.

## Quick Start

1.  **Clone the project** to your local machine.
2.  **API Key:** Ensure your Pexels API key is present in `index.js`.
3.  **Launch:** Open `index.html` in any modern web browser (Chrome or Edge recommended).
